# SecureLogin

一套代码多端组件库方案 Demo 。

## 安装依赖

```shell
pnpm install
```

## 构建基础 `SecureLogin` 开发应用

```shell
npm run dev
```

```shell
pnpm run link
```

## Vue 示例

```shell
# example/vue-example

pnpm install

pnpm link securelogin

npm run serve
```

## React 示例

```shell
# example/react-example

pnpm install

pnpm link securelogin

npm run start
```
